Speech Understanding course syllabus at IIT Jodhpur — B.Tech AI, semester 8.
Topics include the source filter model of speech production, phonetics, phonology, phonemes, allophones, vowels, consonants, nasals, fricatives, stops, plosives, affricates, liquids, glides.
Acoustic features: mel-frequency cepstral coefficients MFCC, linear frequency cepstral coefficients LFCC, constant-Q cepstral coefficients CQCC, perceptual linear prediction PLP, filterbank energies, spectrogram, mel-spectrogram, log-mel, cepstrum, pitch, fundamental frequency F0, formant, harmonic, jitter, shimmer.
Signal processing: pre-emphasis, windowing, Hamming, Hann, short-time Fourier transform STFT, discrete cosine transform DCT, discrete Fourier transform DFT, fast Fourier transform FFT, zero-crossing rate, energy, root mean square RMS.
Probabilistic models: hidden Markov model HMM, Gaussian mixture model GMM, Viterbi decoding, Baum-Welch, forward-backward algorithm, expectation maximization EM, maximum likelihood, Bayesian, posterior, prior, likelihood, stochastic.
Neural speech models: connectionist temporal classification CTC, encoder decoder, attention, transformer, wav2vec, HuBERT, Whisper, conformer, self-supervised learning, contrastive predictive coding CPC, discrete latent, masked language model, fine-tuning, transfer learning.
Speech tasks: automatic speech recognition ASR, speech-to-text STT, text-to-speech TTS, speaker recognition, speaker verification, speaker diarization, language identification LID, voice activity detection VAD, keyword spotting, spoken language understanding SLU, emotion recognition, code switching, code mixing.
Synthesis and vocoders: Tacotron, FastSpeech, WaveNet, WaveGlow, HiFi-GAN, WORLD vocoder, Griffin-Lim, mel-spectrogram inversion, VITS, YourTTS, MMS, Tortoise, XTTS, Bark.
Speaker embeddings: i-vector, x-vector, d-vector, ECAPA-TDNN, speaker encoder, cosine similarity, verification trial, equal error rate EER, detection cost function DCF, ROC curve, DET curve.
Countermeasures and anti-spoofing: bona fide, spoof, replay attack, logical access, physical access, tandem detection cost function t-DCF, ASVspoof challenge, anti-spoofing, deepfake.
Adversarial audio: adversarial example, fast gradient sign method FGSM, projected gradient descent PGD, epsilon perturbation, signal-to-noise ratio SNR, imperceptibility, robustness, defense, gradient masking, universal perturbation.
Prosody: intonation, stress, rhythm, pitch contour, energy contour, duration, speaking rate, voice quality, breathy, creaky, whispered, shouted.
Dynamic time warping DTW, edit distance, warping path, cost matrix, local constraints, Sakoe-Chiba band, Itakura parallelogram.
Evaluation metrics: word error rate WER, character error rate CER, phoneme error rate PER, BLEU, chrF, mean opinion score MOS, perceptual evaluation of speech quality PESQ, short-time objective intelligibility STOI, mel-cepstral distortion MCD, scale-invariant signal-to-distortion ratio SI-SDR, signal-to-noise ratio SNR.
Multilingual and low-resource: code switching, Hinglish, Hindi, English, bilingual, monolingual, multilingual, cross-lingual, zero-shot, few-shot, low-resource language LRL, Santhali, Maithili, Bhojpuri, Awadhi, Gondi, Munda, Dravidian, Indo-Aryan.
Phonology and IPA: International Phonetic Alphabet, grapheme to phoneme G2P, phoneme to grapheme, schwa, retroflex, aspirated, voiced, unvoiced, dental, alveolar, palatal, velar, glottal, bilabial, labiodental.
Voice activity detection VAD, speaker diarization, clustering, agglomerative hierarchical clustering AHC, spectral clustering, variational Bayes, PLDA probabilistic linear discriminant analysis.
Encoder architectures: convolutional neural network CNN, recurrent neural network RNN, long short-term memory LSTM, gated recurrent unit GRU, bidirectional, attention mechanism, self-attention, multi-head attention, positional encoding, layer normalization, residual connection, dropout, batch normalization.
Machine learning fundamentals: gradient descent, Adam, SGD, learning rate, loss function, cross entropy, mean squared error, regularization, overfitting, underfitting, bias variance tradeoff, validation, cross validation.
Speech corpora: LibriSpeech, Common Voice, VCTK, LJSpeech, VoxCeleb, TIMIT, WSJ, TED-LIUM, Fisher, CALLHOME, Switchboard, FLEURS, MUSAN, MLS, People's Speech.
Front end processing: noise suppression, dereverberation, echo cancellation, spectral subtraction, Wiener filter, DeepFilterNet, RNNoise, beamforming, microphone array, source separation.
Hinglish and code switching: matrix language, embedded language, Myers-Scotton, intra-sentential, inter-sentential, tag switching, romanization, transliteration, language identification.
Attention is all you need, encoder-only, decoder-only, sequence-to-sequence, beam search, greedy decoding, constrained decoding, logit bias, lexical shortlist, language model fusion, shallow fusion, deep fusion.
Information theory: entropy, perplexity, Kullback-Leibler divergence, mutual information, channel capacity, source coding, channel coding.
Phoneme set covers stop consonants p b t d k g, fricatives f v s z sh zh, affricates ch j, nasals m n ng, liquids l r, glides w y, vowels a e i o u, diphthongs, monophthongs.
The course evaluates students on programming assignments involving code-switched automatic speech recognition, language identification, voice cloning with VITS or YourTTS, anti-spoofing countermeasures with LFCC, adversarial robustness with FGSM, dynamic time warping based prosody transfer, and evaluation on metrics WER MCD EER.
